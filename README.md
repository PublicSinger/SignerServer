# SignerServer
a simple sign server  
pr and issue are welcome.

## Deploy
* install pythpn 3.12+
* install `libx11-6 libx11-xcb1 libuuid1 zlib1g libxext6 libssl3 libuv 1libnode-dev radare2 patchelf` (ubuntu)
* run `pip install -r requirements.txt`
* run `python3 main.py`
